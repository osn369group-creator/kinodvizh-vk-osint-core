# GitHub-репозитории для изучения VK OSINT

Список направлений, которые можно изучать для безопасной работы с ВКонтакте.

## Основные направления

1. Официальная схема VK API.
2. Клиенты VK API для Python, .NET, Go, PHP.
3. Примеры чтения публичных стен и медиа.
4. Экспорт и структурирование публичных материалов.
5. Проверка ссылок, вложений и времени публикации.

## Что изучать

- Авторизация через официальные методы.
- Ограничения API и rate limits.
- Методы работы с публичными группами и стенами.
- Структуры ответов: post, owner, attachments, photos, videos, comments.
- Обработка ошибок и недоступных данных.

## Что не изучать

- Обход приватности.
- Сбор закрытых данных.
- Токены, cookie, украденные сессии.
- Парсинг закрытых профилей.
- Деанонимизация и пробив.

## Примеры поисковых запросов GitHub

```text
vkontakte api
vk api python
vk public wall api
vk group posts parser
vk wall reader
vk api schema
```

## Правило отбора

Если репозиторий не связан с ВКонтакте, VK API, публичными постами или медиа VK — игнорировать. Например, MoltenVK, VkFFT и Vulkan-проекты не относятся к этой базе.
