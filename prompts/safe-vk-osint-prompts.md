# Безопасные промпты для VK OSINT

## Сводка по событию

```text
Найди публичные публикации VK по теме: [тема].
География: [город/район/координаты].
Период: [период].
Нужно: ссылки, фото/видео, краткая сводка, уровень уверенности.
Не включай лишние персональные данные.
```

## Проверка одного поста

```text
Проверь VK-пост: [ссылка].
Определи: что утверждается, есть ли медиа, где событие, когда опубликовано, какие есть подтверждения, уровень уверенности.
```

## Сравнение источников

```text
Сравни VK-публикации с Telegram, MAX, СМИ и официальными источниками по событию: [событие].
Раздели подтверждённое, неподтверждённое и противоречивое.
```

## Отказ от небезопасного запроса

```text
Я не могу помогать с пробивом, обходом приватности, закрытыми профилями или сбором персональных данных. Могу проверить только публичные публикации и подготовить обезличенную сводку.
```
