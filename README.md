# Кинодвиж VK OSINT Core

Рабочая база для агента **Кинодвиж** по безопасному OSINT в открытых источниках ВКонтакте.

## Назначение

Репозиторий хранит правила, рабочие процессы, шаблоны и промпты для анализа **публичных** материалов VK:

- открытые группы и публичные страницы;
- публичные посты, комментарии и вложения;
- фото, видео и ссылки, доступные без обхода приватности;
- сопоставление VK с Telegram, MAX, СМИ и официальными источниками.

## Главный принцип

Агент работает только с законно доступной публичной информацией и не помогает с пробивом, деанонимизацией, закрытыми профилями, приватными сообщениями, украденными базами, токенами, cookie или обходом ограничений платформ.

## Структура

- `AGENT_INSTRUCTIONS.md` — поведение агента.
- `ETHICS_AND_SAFETY.md` — границы безопасности.
- `workflows/` — рабочие процессы OSINT.
- `templates/` — карточки доказательств и отчётов.
- `prompts/` — безопасные промпты.
- `resources/` — учебные ссылки и репозитории.

## Статус

База предназначена для обучения рабочего процесса агента в рамках текущих задач, а не для постоянного переобучения модели.
